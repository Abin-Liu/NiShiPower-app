工作原理

倪石之力是一个能够自动与魔兽世界《熊猫人之谜》中的NPC“农夫倪石”反复进行宠物战斗，通过战胜NPC的宠物而获得大量经验，使玩家自得的低级宠物快速升级的外挂程序，其工作原理如下：

  1. 自动使用预先选定的宠物进行组队，并设置出最优化的宠物技能与组合。
  2. 与NPC交互并开始宠物战斗。
  3. 在战斗过程中自动选择最适合当前战况的技能以达到理想的战术目的。
  4. 在最恰当的时机使用自爆术，与敌方最后一只宠物同归于尽，使玩家的低级宠物能独享全部经验。
  5. 战斗胜利后自动放弃已完成的任务，回到步骤1。

系统需求

  • Microsoft Windows 2000或更高版本的Windows系列操作系统。
  • 魔兽世界游戏。

安装与运行

  1. 关闭魔兽世界游戏。
  2. 运行下载的倪石之力安装程序。
  3. 启动倪石之力，软件会在系统托盘区添加一个图标。
  4. 运行魔兽世界，进入“按键设置”界面。
  5. 在“按键设置”界面找到“倪石之力”，为“主动作控制”进行按键绑定。如下图所示：

  6. 在“按键设置”界面找到“移动按键”，记住“前进”功能使用的按键。另外找到“目标”，记住“与目标互动”功能使用的按键，如果尚未绑定，请绑定一个按键。如下图所示：

    7.  

  7. 返回Windows桌面，右键点击托盘区的倪石之力图标，在弹出菜单进行选“按键设置”，在各个项目中填入与游戏中相一致的按键。如下图所示：

  8. 进入游戏后在屏幕中间找到“倪石之力”的标题按钮，你可以用鼠标把它拖动到任何地方。点击这个按钮将打开插件主界面，如下图所示：

  9. 点击“倪石之力”插件界面的“幼宠”标签，将你希望练级的低级宠物从宠物手册中拖到插件中。你可以拖入多个宠物，练级顺序为从上至下。如下图所示：

  10. 点击“倪石之力”插件界面的“战宠1”标签，将你所拥有的所有满级的水黾类宠物拖到插件中。
  11. 点击“倪石之力”插件界面的“战宠2”标签，将你所拥有的所有满级的拥有自爆技能的宠物（暗月飞艇、迷你地狱火、步行炸弹、机械熊猫人幼龙等）拖到插件中。
  12. 前往四风谷，靠近NPC“农夫倪石”，确定你所处的位置与NPC足够近（可以直接用鼠标右键交谈）。
  13. 在游戏中按下“Pause”键，程序开始运行。

常见问题及解决方法

  1. 人物站在那里发呆，不进行任何动作。


    2. 答：请正确设置各个快捷键，并确认角色已站在NPC近旁，并检查是否已将各相应宠物拖入插件界面，以及这些宠物是否处于等待复活状态。

  2. 我的程序卡在放弃任务这一步了。


    3. 答：目前游戏确实存在某些任务无法放弃的bug，而这个NPC的任务更是容易产生此bug。遇到这种情况没有办法，只能耐心等待，一般过半小时左右bug会自动消失。

  3. 我插件列表中的宠物已全部满级了，但是程序为什么不能自动接着练手册中的其它低级宠物呢？


    4. 答：我不打算让程序过于随心所欲地左右用户的习惯，也许用户特意想要将某些宠物保持在低等级状态呢？一切皆有可能。

  4. 我会因为使用这个程序而被封号吗？


    5. 答：想听套话吗？请看下面的免责声明。
